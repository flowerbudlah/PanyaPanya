[개인 프로젝트] 팡야팡야
2021년 여름, 프로그래밍에 대해 잘 알지 못하던 시절 실력을 키우기 위해 Spring MVC를 학습하며 단독으로 포트폴리오용 프로젝트를 시작했습니다.
베이커리 쇼핑몰이라는 콘셉트로 게시판과 장바구니 기능 등을 갖춘 웹사이트를 구현했으며, 당시에는 배포 방법을 몰라 완성 후 잠시 멈춰두기도 했습니다.
이후 꾸준히 유지보수와 기능 개선을 진행했고, 지금은 학습과 성장의 흔적이 담긴 의미 있는 프로젝트로 다시 정리해 소개하고자 합니다.
실제 결제 기능(PayPal 등)은 없지만, 기본적인 쇼핑몰 구조와 기능을 갖춘 웹사이트입니다.
